# System Design - Shorter URL

### Functional Requirements
* Given long URL : Shortern it and return to the short URL
* IF someone hits the short url, translate back to the long URL
* IF the same large URL is entered multiple times, need to return a different and unique short URL each time.
### Additional Functional Requirements 
* URLs' can be customized
* TTL (Time To Live) for mappiing
* Analytics
